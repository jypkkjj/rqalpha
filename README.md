# 使用socket5安装

> 前提是你有代理，且本地代理支持socket5服务

1. 先安装pysocks模块

```python
pip install pysocks
```

2. 代理安装安装wxpython
```python 
pip install -U jqdatasdk  --proxy='socks5://127.0.0.1:1081'
```
